Write a program to sort an array of number using Insertion Sort.

<h3>Input Format</h3>
<pre>
	The first line of input specifies the number of test cases, t.
	Each test case consists of the length of the array, n, followed by n integers.
</pre>

<h3>Output Format</h3>
<pre>
	Print the sorted elements for each test case on a single line.
</pre>

<h3>Example Input</h3>
<pre>
	3
	1
	5
	3
	3 11 2
	6
	12 33 2 14 8 1
</pre>

<h3>Example Output</h3>
<pre>
	5
	2 3 11
	1 2 8 12 14 33
</pre>
